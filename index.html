Research Interests

My research interests are in advancing algorithms and systems for extracting structured information from unstructured text.

I'm interested in scalable learning-based approaches to information extraction and knowledge acquisition for knowledge bases.

I'm always interested in applications of these techniques for actionable inferences in a variety of domains.
Currently I'm working on a system to mine social media documents for evidence of foodborne illness outbreaks in restaurants in NYC.

As an undergrad I worked on creating machine learning-enabled web crawlers for smarter information retrieval, open source visualization and analysis software for geospatial data, and theoretically secure fingerprint verification algorithms.

Projects

Foodpoisoning on Social Media

Many people report food poisoning from restaurants on social media, such as Yelp and Twitter.

This project aims to mine these social media streams to identify evidence of restaurants in NYC that may be the sources of foodborne illness outbreaks. We work with the NYC Department of Health to use this software to help launch investigations.

FindIt: Smart Web Crawling

Often different websites host many pages that share a common theme, but have non-uniform structure; think college course descriptions.

FindIt uses semi-supervised machine learning to quickly learn search strategies that generalize across website organizational structures.

Fingerprints as Passwords

This project was at the intersection of cryptography, biometrics, and pattern recognition.

The goal was to develop theoretically secure encryptions of fingerprint templates such that matching was still possible in the encrypted space.

iGlobe: Geospatial Machine Learning

iGlobe is a machine learning layer built on top of NASA's WorldWind. It allows for the incorporation, visualization, and analysis of independent geospatial datasets.

I worked on this project with my machine learning professor, Dr. Varun Chandola for the NASA Europa Challenge, where our team won 2nd place for university entries.
